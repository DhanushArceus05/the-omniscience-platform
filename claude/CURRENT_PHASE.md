# Phase 0 — Foundation

## Deliverables

- pnpm workspace + Turborepo
- apps/web: React + Vite + TypeScript
- apps/api: NestJS + TypeScript
- apps/ai-service: FastAPI
- shared packages for ui, types, schemas, config, sdk, prompts and utils
- Docker Compose for PostgreSQL, MongoDB, Redis and Qdrant
- env validation and `.env.example`
- lint, format, strict typing
- health endpoints
- structured logging and global error handling
- starter tests
- GitHub Actions CI
- root README
- approved docs copied into repository

## Exclusions

No auth, workspaces, AI calls, OmniCore, RAG, agents, ML features or final product screens.
