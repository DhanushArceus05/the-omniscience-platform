# AI Architecture

Version: 1.0
Status: Approved
Last Updated: 2026-07-11

- OmniCore: routing, planning, orchestration, validation and fallback
- OmniProvider: provider plugin adapters
- OmniModel Manager: quality, latency, cost, context, health and privacy-aware selection
- OmniKnowledge: parsing, chunking, embeddings, hybrid retrieval, reranking, citations and confidence
- OmniMemory: session, workspace, user, knowledge and agent memory with confidence, expiry and controls
- OmniAgents: chief, planner, reviewer and specialist agents
- OmniPredict: classical ML, forecasting, recommendations, anomaly detection, AutoML and XAI
- OmniVision: OCR, visual understanding, object detection and document/chart/table/diagram analysis
- OmniVoice/OmniLingo: STT, TTS, voice commands, language detection and translation
- OmniFlow: visual workflows, triggers, retries, parallelism and replay
